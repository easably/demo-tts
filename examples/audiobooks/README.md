# Demo audiobook filenames

Pattern: `{scope}-{engine}-{voices}-{mode}.mp3`

- **scope**: `full-book` | `part-01`
- **engine**: `tada` | `dots`
- **voices**: `one-voice-owen` | `multi-voice-cast`
- **mode**: `chunks` (single narrator) | `directed` (production / role cast)

See [the-red-headed-league/a1/README.md](./the-red-headed-league/a1/README.md) for this book’s files.
